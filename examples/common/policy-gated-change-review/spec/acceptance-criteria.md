# Example Spec: Acceptance Criteria

The running example is accepted when the following conditions hold.

## Acceptance Criteria

- The change request, review plan, and approved change are explicit artifacts.
- Policy checks and human approval are distinct steps.
- The artifact path from request to approval is traceable.
- The example can be referenced from specification, design, verification, and implementation discussions.
