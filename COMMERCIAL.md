# Commercial Use Guidance

This file covers commercial-use guidance for the repository's dual-license model.
This file is a repository policy statement and not legal advice.

## Book content

Reader-facing book content is licensed under `CC BY-NC-SA 4.0`.
That default license does not grant commercial use of the book prose, figures, or editorial structure.

If you need commercial rights for book content, contact ITDO Inc. at `info@itdo.jp`.
Do not post confidential commercial terms in a public GitHub issue.

## Code and reusable technical assets

Code, build files, schemas, automation, site-generation assets, and reusable technical examples are licensed under `Apache-2.0`.
That license already permits commercial use, subject to its own conditions.

## Trademarks and third-party assets

Commercial rights to book content do not include trademark rights.
Third-party and upstream materials remain subject to their own licenses and notices.

See [`LICENSE-SCOPE.md`](LICENSE-SCOPE.md), [`TRADEMARKS.md`](TRADEMARKS.md), and [`THIRD_PARTY_NOTICES.md`](THIRD_PARTY_NOTICES.md) for the canonical boundary.
